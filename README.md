# Encoder Application

A lightweight Java-based Application encoding and decoding data.
This project is designed for educational purposes and extensibility into a more
robust encoding service.

## Tech Stack

- Java 23
- Maven
- Spring Boot
- PostgreSQL for Database
- JUnit 5 for Testing
